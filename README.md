---

<h2>📌 1. O que é o GitHub?</h2>
<p>GitHub é uma plataforma para alojar código, colaborar em projetos e gerir versões usando Git.</p>

<ul>
  <li>Guarda o histórico de alterações</li>
  <li>Permite trabalho em equipa</li>
  <li>Suporta projetos públicos e privados</li>
</ul>

---

<h2>🧰 2. Criar uma conta</h2>
<p>Para começar, acede a <strong>github.com</strong> e cria uma conta gratuita.</p>

</p>

<h2>📦 3. Criar o primeiro repositório</h2>
<p>Um repositório é onde o teu projeto vive.</p>

<ul>
  <li>Clica em <strong>New Repository</strong></li>
  <li>Escolhe um nome</li>
  <li>Define se é público ou privado</li>
  <li>Adiciona um README (opcional mas recomendado)</li>
</ul>

<p align="center">
  <img src="imagens/novo-repo.png" width="600">
</p>

<h2>🖥️ 4. Clonar o repositório</h2>
<p>Para trabalhar no teu computador:</p>
<h2>📝 5. Fazer commits</h2><p>Um commit regista uma alteração no projeto.</p>
git add .
git commit -m "Mensagem do commit"
git push
<h2>🔀 6. Criar um Pull Request</h2><p>Um Pull Request serve para propor alterações ao projeto.</p>
<ul><li>Cria uma branch</li><li>Faz alterações</li><li>Abre um Pull Request</li><li>Espera pela revisão</li></ul>

<h2>📣 7. Issues</h2><p>Issues servem para reportar bugs, pedir funcionalidades ou organizar tarefas.</p>

<h2>🚀 8. GitHub Pages</h2><p>Podes publicar um site diretamente a partir do repositório.</p>
<ul><li>Vai a <strong>Settings → Pages</strong></li><li>Escolhe a branch</li><li>Publica</li></ul>
